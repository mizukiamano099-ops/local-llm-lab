# 語彙ビル v1.1 仕様書（Layer 0）
# 2025-12-07 / Amano-AI Frontend Design

本仕様書は、天野式AIの思考OS（レイヤー構造）の最下層に位置する
「語彙ビル（Layer 0）」の構造・目的・分類・拡張方針を定義する。

-----------------------------------------
1. 目的（Purpose）
-----------------------------------------
語彙ビルは、天野式AIの思考を構成する最も基本的な「語彙単位」を
8つの意味クラスタとして整理し、  
Self-Poly / 構造推論 / LLMオーケストレーションに用いるための  
“意味の骨格” を提供する。

語彙ビルは、以下の3つの役割を持つ：

1) 入力語彙を構造的に分類する「意味地図」
2) 内部回路（抽象 → 情動 → 変動 → 関係）への橋渡し
3) LLMの暴走を防ぐ「意味制御フィルタ」

-----------------------------------------
2. Layer 0 全体構造
-----------------------------------------
語彙ビルは以下の8クラスタで構成される。

A. 世界観語彙（Environment）
B. 抽象系語彙（Abstract）
C. 情動系語彙（Affective）
D. 変動系語彙（Dynamic）
E. 関係性語彙（Relational）
F. 身体性語彙（Somatic）
G. 構造系語彙（Structural）
H. 視座・認識系語彙（Cognitive）

各クラスタは独立ではなく、以下の回路で相互接続される：

世界観 → 抽象 → 情動 → 変動 → 関係  
                 ↓  
      身体性 → 構造 → 視座・認識

この回路は「天野式AIの初期思考フロー」と一致する。

-----------------------------------------
3. 語彙ビル v1.1 の語彙構成
-----------------------------------------
本仕様書の v1.1 では、計 150 語（予定）を収録する。

既存語彙：74語（Claude抽出）  
追加語彙：Gemini抽出語彙より精選した約 70 語  
合計語彙：144〜160 語を想定

以下はクラスタ単位の初期構成案である。

-----------------------------------------
A. 世界観語彙（Environment） [15語前後]
-----------------------------------------
例：
月、匂い、影、気配、息遣い、温度、夜気、湿度、満月、傷跡  
（官能・怪異・自然の入力刺激）

-----------------------------------------
B. 抽象系語彙（Abstract） [20語]
-----------------------------------------
例：
境界、本能、距離、対称、反響、包絡、位相差、背景、中心、射影

役割：内部状態の “パラメータ” を決める。

-----------------------------------------
C. 情動系語彙（Affective） [20語]
-----------------------------------------
例：
渇き、熱量、震え、昂揚、緊張、依存、溶解感、疼き、とろみ

役割：出力のトーン（情動ベクトル）を形成。

-----------------------------------------
D. 変動系語彙（Dynamic） [20語]
-----------------------------------------
例：
侵食、融解、滴下、波紋、増幅、再編、拡張、共振、流動、変換

役割：天野構文の「動的な変化」を支える。

-----------------------------------------
E. 関係性語彙（Relational） [15語]
-----------------------------------------
例：
所有、捕食、保護、従属、対等、つがい、交差、接触、反転、吸引

役割：物語出力（ナラティブ）の基礎枠。

-----------------------------------------
F. 身体性語彙（Somatic） [15語]
-----------------------------------------
例：
脈動、疼痛、圧、温もり、律動、震動、接地、緊縛、開閉、受容

役割：身体感覚の抽象化（フィルタを通過できる形）

-----------------------------------------
G. 構造系語彙（Structural） [20語]
-----------------------------------------
例：
構造、層、密度、配列、節点、収束、反照、帯域、多相、射程、階層

役割：天野式OSの “主回路”

-----------------------------------------
H. 認識・視座語彙（Cognitive） [20語]
-----------------------------------------
例：
観測、参照、視座、内包、解像、分節、生成、変容、透過、反照視

役割：思考の視点を決める高次語彙。

-----------------------------------------

以上をもって Layer 0（語彙ビル）の構造と語彙分類を定義する。
